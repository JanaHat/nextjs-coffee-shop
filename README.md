# Coffee Product Catalogue (Next.js App Router)

A small product catalogue built with Next.js App Router + TypeScript.

Domain is coffee, with search/filter/sort, product detail, and basket flow.

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Route Handlers for local API
- Redux

## Setup

### Prerequisites

- Node.js 20+
- npm 10+

### Install and run

1. Install dependencies:

`npm install`

2. Start development server:

`npm run dev`

3. Open:

`http://localhost:3000`

### Useful scripts

- `npm run lint`
- `npm run format`
- `npm run format:check`
- `npm run build`
